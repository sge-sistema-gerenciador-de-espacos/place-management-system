# place-management-system

## Getting started

```bash
# clone the project
# if uses ssh
git clone git@github.com:sge-sistema-gerenciador-de-espacos/place-management-system-server-side.git
# if uses https
git clone https://github.com/sge-sistema-gerenciador-de-espacos/place-management-system-server-side.git

# enter the project directory
cd place-management-system-server-side

# run project
# if uses windows
gradlew bootRun
# if uses unix
./gradlew bootRun
```
